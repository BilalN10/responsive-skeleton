Widget

Widget Topic

Classes

BasicWebParentPage Widget
Widget par défaut, permet d'instancier une page de largeur maximum et d'ajouter la sidebar (uniquement si AppConfigModel.showSideBarOrDrawerOnWeb est à true)
ClampingPage Widget Page
Widget parent créant une page de taille fixe (MediaQuery), à mettre avant votre page, pour gérer les différentes fonctionnalités de vote interface. Va générer automatiquement la bottom bar et la sidebar.
CustomCard Widget
Créer une card (Équivalent de Card de flutter) avec un style personnalisé et entièrement customisable.
CustomExpansionPanelList Widget
Widget de liste déroulante
ImageWithSmartFormat Widget
Créer automatiquement une image avec le type demandé. Si l'image est un url internet, ajout automatique du cache.
ResponsiveAppBar Widget
Créer une AppBar avec un bouton de profil et les boutons de navigation (uniquement si configuré dans ApplicationModel).
ScaffoldWithLogo Widget
Widget créant un scaffold avec le logo de l'application (path défini dans ApplicationModel). Le logo est centré dans le cas d'une taille d'écran inférieur à une tablette ou dans une application mobile. Sinon, il est par défaut positionner en Alignment.topLeft
ScrollingPage Widget Page
Widget parent permettant un scrolling par défaut, à mettre avant votre page, pour gérer les différentes fonctionnalités de vote interface. Va générer automatiquement la bottom bar et la sidebar.
WebParentCore Widget
Class contenant les différentes méthodes de création d'interface. (Ex. buildBottomBar => bottomBar si active dans le model, buildLeftSideBar => sideBar, contenant les liens de navigation, si active dans le model)
